# CSRF-testing
CSRFを試したリポジトリ
起動して、以下のURLにアクセスすると挙動が確認可能(シークレットブラウザがおすすめ)

http://localhost:3000/1-get.html

http://localhost:3000/2-post.html

http://localhost:3000/3-cookie.html

http://127.0.0.1:4000/login

http://localhost:3000/4-cookie-xhr.html

# 起動方法
node,npmが使える環境が必要。
```bash
npm i
npm start
```
